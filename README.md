# ApiEnderecoClient

## Como roda o projeto?

```
Entre no diretorio onde está localizado o projeto angular e rode os seguintes comandos:

npm install
ng serve


- Pronto!! Seu client estará disponível na porta 4200. ex: localhost:4200

```
